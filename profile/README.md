## 🦺 현장의 모든 순간을 담다, Build-Up
> 건설현장 노무관리를 위한 AI 플랫폼

Build-Up은 “**건설현장**의 **근로계약–근태–급여–안전**을 플랫폼·AI 기반으로 자동화하여 체계적이고 투명한 **노무 관리** 환경을 구축”하는 통합 솔루션이에요. <br/>
빌드업 솔루션은 기업/현장 관리자용 웹, 근로자용 웹앱, 얼굴인식 출퇴근 시스템, 안전장비(PPE) 착용여부 감지 AI 모델을 제공해요.

<br>

- [🧑🏻‍💼 기업/현장 관리자용 웹 링크](https://bu-manager-web.vercel.app)
- [🦺 근로자용 웹앱 링크](https://bu-worker-web.vercel.app)
- [👷🏻 출퇴근 시스템 웹앱 링크](https://bu-site-web.vercel.app)
- [🎬 서비스 시연 영상 링크](https://youtu.be/QODWwUFJvxM)
